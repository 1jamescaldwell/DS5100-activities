Exercise 3.1

val = 5

if -10 < val < - 5:
    print('bucket 1')
elif -5 <= val <= -2:
    print('bucket 2')
elif val == -2:
    print('bucket 3')

If val = 5, nothing will print.

To fix the problem, add another elif condition if needed, or an else:

val = 5

if -10 < val < - 5:
    print('bucket 1')
elif -5 <= val <= -2:
    print('bucket 2')
elif val == -2:
    print('bucket 3')
else:
    print('bucket 4')
bucket 4
